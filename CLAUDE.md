## Project Overview

**Myzel** is a custom Infrastructure as Code (IAC) framework in Python. 
It allows you to define AWS resources as Python classes, 
automatically detect differences between desired and deployed state, 
and apply changes.

**Key Philosophy**: Simple happy path programming till it brakes.
